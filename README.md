# Web Base

## Descrição

Base de página de banco

## Começando

### Dependências

* [pnpm](https://pnpm.io/pt/)
* [Vite & React](https://vitejs.dev/)

### Instalação

* Clone o projeto e abra no terminal, após digite o comando abaixo
```
pnpm install
```

### Executando

* Digite o comando abaixo
```
pnpm dev
```
